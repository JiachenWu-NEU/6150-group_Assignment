# 6150-group_Assignment

## Project Name: TradeHub

## Authors: Jiachen Wu, Rui Sun, Yunqi Zhang, Zehao Wu, Tiantong Zhang

- Description

    - A lightweight registration page for a second-hand marketplace where buyers discover local deals and sellers list pre-loved items with ease. It features clean Bootstrap styling, live HTML5 validation, and a password strength meter to promote trust and safety. On successful submit, the form shows a concise summary so users can quickly proceed to buying or selling.

- HTML Tags Used

    - `div` `h1` `p` `form` `label` `input` `type` `span` `i` `button` `a` `progress`

- js techniques

    - validation(using regular expressions), strength meter, toggle password, card selection, submit alert

- others

    - using bootstrap for beautifying pages
